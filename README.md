## Getting Started

### General info
Companies Review application. Technologies: React, API and Bootstrap

### Prerequisites

* Node js
```sh
https://nodejs.org/dist/v12.15.0/node-v12.15.0-x64.msi
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/krzosdominik/companies-review.git
```
2. Install NPM packages
```sh
npm install
```
3. Run app
```sh
npm start
```
